### Configuración de Base de Datos PostgreSQL

# 📦 Creación de la Base de Datos

```sql
-- Database: restaurant_db

-- DROP DATABASE IF EXISTS restaurant_db;

CREATE DATABASE restaurant_db
    WITH
    OWNER = postgres
    ENCODING = 'UTF8'
    LC_COLLATE = 'Spanish_El Salvador.1252'
    LC_CTYPE = 'Spanish_El Salvador.1252'
    LOCALE_PROVIDER = 'libc'
    TABLESPACE = pg_default
    CONNECTION LIMIT = -1
    IS_TEMPLATE = False;
```

# 🍛 Tabla `plate`

## 🛠️ Estructura de la Tabla

```sql
-- Table: public.plate

-- DROP TABLE IF EXISTS public.plate;

CREATE TABLE IF NOT EXISTS public.plate
(
    id bigint NOT NULL GENERATED BY DEFAULT AS IDENTITY 
    (
        INCREMENT 1 
        START 1 
        MINVALUE 1 
        MAXVALUE 9223372036854775807 
        CACHE 1
    ),

    plate_description character varying(255) 
        COLLATE pg_catalog."default",

    plate_is_available boolean,

    plate_name character varying(255) 
        COLLATE pg_catalog."default",

    plate_price double precision,

    CONSTRAINT plate_pkey PRIMARY KEY (id)
)

TABLESPACE pg_default;

ALTER TABLE IF EXISTS public.plate
    OWNER to postgres;
```

---

# 📋 Descripción de Campos

| Campo | Tipo de Dato | Descripción |
|---|---|---|
| `id` | `bigint` | Identificador único del plato |
| `plate_description` | `varchar(255)` | Descripción del plato |
| `plate_is_available` | `boolean` | Disponibilidad del plato |
| `plate_name` | `varchar(255)` | Nombre del plato |
| `plate_price` | `double precision` | Precio del plato |
